# 🚀 SEMD Token Platform

מערכת מתקדמת למסחר והשקעה במטבע SEMD על רשת סולנה, המשלבת בינה מלאכותית ואנליטיקה מתקדמת.

## 🌟 תכונות עיקריות

### 💹 מסחר חכם
- בוט מסחר אוטומטי עם בינה מלאכותית
- זיהוי מגמות שוק בזמן אמת
- ניהול סיכונים מתקדם
- אסטרטגיות מסחר מותאמות אישית

### 🔗 טכנולוגיית בלוקצ'יין
- חוזים חכמים על רשת סולנה
- אבטחה מקסימלית
- עמלות נמוכות
- מהירות עסקאות גבוהה

### 📊 אנליטיקה
- דשבורד מתקדם לניתוח ביצועים
- גרפים בזמן אמת
- התראות מותאמות אישית
- דוחות מפורטים

## 🛠 התקנה

### דרישות מערכת
- Python 3.8+
- Node.js 16+
- Rust (לפיתוח חוזים חכמים)

### שלבי התקנה

1. התקנת dependencies:
```bash
pip install -r requirements.txt
cd src/frontend && npm install
```

2. הגדרת משתני סביבה:
```bash
SOLANA_RPC_URL=<your-rpc-url>
SEMD_TOKEN_ADDRESS=<token-address>
API_KEY=<your-api-key>
```

3. הפעלת המערכת:
```bash
# הפעלת הבוט
python src/bot/trading_bot.py

# הפעלת הממשק
cd src/frontend && npm start
```
